DESCRIPTION


In this code assignment, I was instructed to apply my prior knowledge of arithmetic operators to perform calculations based on the user’s inputs. This program acts as a calculator that allows the user to input an integer (no decimals, or floats) to perform either multiplication, addition, division or subtraction.

REFLECTION


In 3.3 Functions Make, I learned how to Define four subroutines - add, subtract, multiply, divide that add multiply etc two numbers and return the result. Each should have two integer number arguments. The user is asked to input two numbers. These numbers will be passed as arguments into one of the subroutines. The user is asked to input 1 to add, 2 to subtract etc. If they input 1, call the ‘add’ subroutine, input 2 calls the ‘subtract’ subroutine etc.
